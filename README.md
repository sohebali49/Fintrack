# 💰 Expense Tracker - MERN Stack

A full-stack Expense Tracker web application built with the **MERN stack**  
(MongoDB, Express.js, React.js, Node.js) to help users track income, expenses, budgets, and spending patterns in one place.

## ✨ Features

- Add, edit, and delete expenses.
- Categorize expenses by type.
- View spending summary on the home dashboard.
- Visualize expenses with charts.
- Track budgets and progress.
- Responsive and user-friendly UI.
- Backend APIs for CRUD operations.
- MongoDB database for storing data.
- Secure environment configuration using `.env`.

## 🛠️ Tech Stack

### Frontend
- React.js
- React Router
- Axios
- Chart.js / Recharts
- CSS / Tailwind CSS

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose

### Tools
- Git & GitHub
- VS Code
- Postman
